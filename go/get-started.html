# Get Started

# Prerequisites

Counter API is a REST API to help developers to keep their count data online. It is design to use in static applications where you do not have backend to store your data.

  • Go (opens new window)

# Quick Start

The fastest way to get Counter API for your project, install it with go itself.

To use it, open up your terminal and run the following command:

go get -u github.com/counterapi/counter
